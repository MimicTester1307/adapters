### QUESTION
What is the value of 'house' + 'abc' + 'xyz'?

### ANSWER
The value of 'house' + 'abc' + 'xyz' is 17.

### EXPLANATION
The calculation is:

* 'house' = 2
* 'abc' = 5
* 'xyz' = 10

So, the sum is: 2 + 5 + 10 = 17.

### CONTEXT
Given 'house' = 2, 'abc' = 5, 'xyz' = 10.

### QUESTION
What is the value of 'house' + 'abc' - 'xyz'?

### ANSWER
The value of 'house' + 'abc' - 'xyz' is 17.

EXPLANATION

Using the given values, we can calculate the sum of 'house' and 'abc' as follows:

'house' = 2 + 'abc' = 5 = 7

Now, we subtract the value of 'xyz' from the sum:

7 - 'xyz' = 10 = 7

Therefore, the value of 'house' + 'abc' - 'xyz' is 7.
--
